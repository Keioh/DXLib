DiploidCamera

void SetMainCameraPosition_DiploidCircleV2(DiploidCircleV2* target)

targetをメインカメラにし、SetMainCameraCenterPosition()で指定した座標に表示します。全てのオブジェクトの中で1つだけ存在することが望ましいです。

  

void SetOtherLookObjectPosition_DiploidCircleV2(DiploidCircleV2* target)

メインカメラに設定しなかった他のオブジェクトをメインカメラから正しく見えるように移動させます。

    
  

void SetMainCameraCenterPosition(VECTOR new_position)

メインカメラの位置を設定します。

    
  

VECTOR GetMainCameraCenterPosition()

現在のメインカメラの位置を取得します。
