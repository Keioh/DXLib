クラス名

関数名

概要

 

DiploidCircleV2

list<VECTOR> hit_points_list

ヒットした場所を保存するためのリスト。このコンテナへのアクセスはなるべく関数を用いること。

  

void Init(VECTOR pos, float radi, unsigned int color)

DiploidCircleを初期化します。posには描画したい位置、radiには円の半径、colorには色を指定します。posにはVGet()関数を使用します。colorにはGetColor()関数を使用します。

  

void MoveUpdata()

SetMoveSpeed()又は、SetMoveAngle()で設定した移動速度を反映します。

  

void Draw(bool draw)

円を描画します。drawにfalseを入れると円の描画がされません。

    
  

void SetName(string new_name)

新しくオブジェクトの名前を設定します。new_nameにはstring型で文字列を入力します。

  

void SetColor(unsingned int new_color)

新しく色を設定します。new_colorにはGetColor()関数を使用します。

  

void SetPosition(VECTOR new_pos)

新しく位置を設定します。new_posにはVGet()関数を使用します。

  

void SetRadius(float new_radius)

新しく円の半径を設定します。new_radiusにはfloat型で数値を入力します。

  

void SetFill(bool new_fill)

新しく塗りつぶしの設定をします。new_fillにはbool型で値を入力します。tureで塗りつぶします。

  

void SetThickness(float new_thickness)

新しく線の太さを設定します。new_thicknessにはfloat型で数値を入力します。1.0fがデフォルトです。

  

void SetObjectNumber(int new_number)

新しいオブジェクト番号を設定します。new_numberにはint型で数値を入力します。0以上の数値が望ましいです。

  

void SetHitFlag(bool new_hit_flag)

新しいヒットフラグを設定します。new_hit_flagにはbool型で値を入力します。trueでヒット状態を表します。

  

void SetMoveSpeed(VECTOR new_move_speed)

新しく移動速度を設定します。new_move_speedにはVECTOR型で数値を入力します。z軸は使用していません。

  

void SetMoveSpeed(float angle, float new_move_speed)

新しく移動速度と移動する角度を設定します。angleにはfloat型で数値を入力します。単位はラジアンです。new_move_speedにはfloat型で移動速度の数値を入力します。

  

void SetLife(float new_life)

新しくオブジェクトの寿命を設定します。

  

void SetDestoryFlag(bool new_flag)

新しく削除フラグを設定します。このフラグをtrueにしてもオブジェクトは削除されません。あくまでも指標として存在します。

    
  

void AddLife(float add_val)

寿命カウントに数値を足します。

  

void SubLife(float sub_val)

寿命カウントから数値を引きます。

    
  

string GetName()

現在のオブジェクトの名前を取得します。戻り値はstring型です。

  

VECTOR GetPosition()

現在のオブジェクトの位置を取得します。戻り値はVECTOR型です。

  

float GetRadius()

現在のオブジェクトの円の半径を取得します。戻り値はfloat型です。

  

unsigned int GetColor()

現在のオブジェクトの色を取得します。戻り値はunsigned int型です。

  

bool GetFill()

現在の塗りつぶし設定を取得します。戻り値はbool型です。

  

float GetThickness()

現在の線の太さの設定を取得します。戻り値はfloat型です。

  

bool GetHitFlag()

現在のオブジェクトのヒットフラグを取得します、戻り値はbool型です。

  

int GetObjectNumber()

現在のオブジェクトの番号を取得します。戻り値はint型です。

  

VECTOR GetMoveSpeed()

現在のオブジェクトの移動している速度を取得します。戻り値はVECTOR型です。

  

float GetMoveAngle()

現在のオブジェクトの移動している方角を取得します。戻り値はラジアンでfloat型で返します。

  

float GetLife()

現在のオブジェクトの寿命を取得します。戻り値はfloat型です。

  

bool GetDestoryFlag()

現在の削除フラグを取得します。戻り値はbool型です。trueで削除フラグがオンです。

    
  

size_t GetHitPositionVolume()

現在の当たっている場所の位置の総数を取得します。戻り値はsize_t型です。

  

VECTOR GetHitPosition()

現在の当たっている場所の位置を取得します。取得されるデータはリストの最初のデータです。戻り値はVECTOR型です。全て-1を返した場合は当たっている場所はありません。

  

list<VECTOR>::iterator GetHitPointsListIterator(list<VECTOR>::iterator* pointer_to_iterator)

この関数の使用は避けてください。hit_poins_listへのイテレータを取得するための関数です。pointer_iteratorへはhit_points_listのイテレータの保存したポインタ変数を指定します。

  

list<VECTOR>::iterator GetHitPointsListIteraltor()

hit_points_listへのイテレータを取得します。取得時の初期値はhit_points_list.begin()です。
